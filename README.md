# Vesta — Landing Page

Landing page da Vesta, servida via GitHub Pages.

Construída com o design system oficial da marca (paleta copper + navy, fonte
Inter, logos SVG oficiais). Single-file, sem dependências além do Google Fonts.

## Deploy

O deploy é automático via GitHub Actions (`.github/workflows/pages.yml`) a cada
push na branch `main`.

Para ativar o GitHub Pages neste repositório:

1. **Settings → Pages**
2. Em **Source**, selecione **GitHub Actions**

A página fica disponível em `https://dev-vesta.github.io/landing_page/`.
